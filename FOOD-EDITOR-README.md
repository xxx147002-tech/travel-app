# 美食清單編輯器使用說明

## 🎯 功能概述

這是一個獨立的美食清單管理工具，讓您可以輕鬆管理旅遊美食資訊，並將資料匯出到主應用程式中使用。

## 📖 使用方式

### 1. 開啟編輯器

在瀏覽器中開啟：`http://localhost:8080/food-editor.html`

### 2. 管理分類

**預設分類**：
- 澎湖市區（紫色）
- 澎湖環島（綠色）

**新增分類**：
1. 在「分類管理」區塊輸入分類名稱
2. 選擇顏色
3. 點擊「新增分類」

**刪除分類**：
- 點擊分類標籤上的 ×

### 3. 新增餐廳

在右側「新增美食」表單中填寫：

| 欄位 | 說明 | 必填 |
|------|------|------|
| 餐廳名稱 | 餐廳的名稱 | ✅ |
| 地址 | 完整地址 | ⭕ |
| 營業時間 | 例：10:00 - 18:00 | ⭕ |
| Google 評分 | 0-5 分，可填小數點 | ⭕ |
| 分類 | 選擇所屬分類 | ✅ |
| Google Maps 連結 | 複製 Google Maps 的分享連結 | ⭕ |

點擊「新增餐廳」完成。

### 4. 編輯與刪除

- **編輯**：點擊餐廳卡片的「編輯」按鈕
- **刪除**：點擊餐廳卡片的「刪除」按鈕

### 5. 匯出資料

1. 點擊「匯出 JSON」按鈕
2. 查看匯出的 JSON 格式資料
3. 點擊「複製到剪貼簿」

### 6. 整合到 APP

**步驟**：

1. 在 `food-editor.html` 完成所有餐廳資料編輯
2. 點擊「匯出 JSON」
3. 點擊「複製到剪貼簿」
4. 打開 `app.js` 檔案
5. 找到這一行：
   ```javascript
   const foodListData = null; // 將從 food-editor.html 匯出的資料貼到這裡
   ```
6. 將複製的 JSON 資料替換 `null`：
   ```javascript
   const foodListData = {
     "categories": [...],
     "foodList": [...]
   };
   ```
7. 儲存檔案

**未來開發**：
- 目前資料結構已預留在 `app.js` 中
- 後續可在主 APP 中新增美食清單顯示功能
- 可依分類篩選顯示

## 💾 資料儲存

- 所有資料自動儲存在瀏覽器的 LocalStorage
- 關閉瀏覽器後資料不會遺失
- 清除瀏覽器資料會導致資料遺失，建議定期匯出備份

## 📊 資料格式

### 分類格式
```json
{
  "id": "unique-id",
  "name": "分類名稱",
  "color": "#顏色代碼"
}
```

### 餐廳格式
```json
{
  "id": "unique-id",
  "name": "餐廳名稱",
  "address": "地址",
  "hours": "營業時間",
  "rating": 4.5,
  "category": "分類ID",
  "mapUrl": "Google Maps 連結"
}
```

## 🎨 特色功能

- ✅ 即時統計顯示
- ✅ 分類顏色自訂
- ✅ 響應式設計（支援手機）
- ✅ 美觀的現代化介面
- ✅ 本地自動儲存
- ✅ 一鍵匯出 JSON
- ✅ 複製到剪貼簿

## ⚠️ 注意事項

1. **資料備份**：定期使用「匯出 JSON」功能備份資料
2. **瀏覽器支援**：建議使用 Chrome、Edge、Safari 等現代瀏覽器
3. **Google Maps 連結**：
   - 在 Google Maps 中找到地點
   - 點擊「分享」
   - 複製連結並貼到編輯器中

## 🚀 範例流程

1. 開啟 `food-editor.html`
2. 建立分類：「馬公市區」、「湖西鄉」
3. 新增餐廳：
   - 名稱：星光海洋牧場
   - 地址：澎湖縣湖西鄉北寮村
   - 營業時間：10:00 - 18:00
   - 評分：4.5
   - 分類：湖西鄉
4. 匯出 JSON
5. 複製到 `app.js`
6. 完成！
